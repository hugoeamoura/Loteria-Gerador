<h1> Projeto - Loteria-Gerador </h1>

<p>O Loteria-Gerador é um peojeto pessoal desenvolvido para atender aqueles que querem 
  jogar números aleatórios na loteria, porém têm dificuldades de decidir os 6 números.
</p>
